# Interview Lab v0.1

自用型面试训练工具：问题预测 + 回答优化 + 深度追问，数据仅存当前会话，无登录、无数据库。

## 环境要求

- Node.js 18+
- OpenAI API Key（用于问题生成、回答优化、追问生成）

## 快速启动

1. 安装依赖（根目录 + server + client）：
   ```bash
   npm run install:all
   ```

2. 配置后端环境变量（在 `server` 目录下创建 `.env`）：
   ```
   OPENAI_API_KEY=sk-xxx
   # 可选：OPENAI_MODEL=gpt-4o-mini
   ```

3. 同时启动后端与前端：
   ```bash
   npm run dev
   ```

4. 浏览器打开：前端 http://localhost:5173 ，后端 API http://localhost:3001

## 使用说明

- **首页**：上传简历（PDF/Word）、输入目标岗位、选择公司风格（落地型/思维型/压力型），点击「开始训练」。
- **训练页**：按题作答，可「提交回答」查看四层优化反馈，或「跳过本题」。若当前题为「项目拆解题」，提交后会出现 6 个追问方向，点选后生成追问，回答追问同样会得到优化反馈。
- 所有数据仅在当前会话内有效，关闭页面即清空。

## 项目结构

- `client/`：React + Vite 前端
- `server/`：Express 后端，简历解析（PDF/Word）、OpenAI 调用、Prompt 模板

## 验收要点（v0.1）

- 首页输入项可操作，开始训练后跳转训练页
- 简历可上传并解析，用于问题预测
- 问题预测输出 5 大类结构化问题，带「为什么会被问」「对应考察能力」
- 回答提交后输出四层分析（结构评分、能力映射、逻辑漏洞、升级示范）
- 项目拆解题后可生成 6 类追问，点选生成追问问题，追问回答可触发优化反馈
- 数据仅存当前会话，关闭页面后清空
